resistance_unifrac {permSEM}R Documentation

Fast Estimation of Slope and Intercept

Description

Calculate the slope and intercept of linear model.

Usage

slope(X,Y,with.intercept=TRUE)

Arguments

X

the x-axis coordinates of a linear model

Y

the y-axis coordinates of a linear model

with.intercept

logical. If TRUE, estimate the interceptof a linear model. Default TRUE
