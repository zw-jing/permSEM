within_dist {permSEM}R Documentation

Extracting the Within Group Distances

Description

Extracting the within group distances and generating a dataframe with mapping columns

Usage

within_dist(dist,group=list())

Arguments

dist

distance matrix generated by vegdist in library vegan

group

A list or a data.frame. Can be multiple levels of group vectors

Author(s)

Zhongwang Jing

Examples

library(vegan)
data(Apis)
subdist=within_dist(bray_dist,map)
